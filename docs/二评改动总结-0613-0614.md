## 二、机构前台 H5（C 端）

**🟢 新增**
- **个人资料页**：头像 / 昵称 / 性别 / 地区可编辑（微信带回 / 非微信自填）；头像相册 + 圆形裁剪；地区省 / 市 / 区级联（真实数据）。
- **我的纸书**：线下纸书二维码扫码绑定的知识 KP 列表（封面 + 名称 + 扫码时间）+ 搜索 + 右上「扫一扫」；点击进入该 KP 全新会话；免费用户也可畅享该 KP 文 / 图 / 音 / 视，永享内容仍另购。
- **敏感操作手机号绑定门槛**：支付 / 兑换 / 开通·充值·续费会员前校验绑定（可暂不绑定）。
- **输入框语音输入**（底部音浪浮层，点浮层外即发送）。
- **答案操作**：复制 / 点赞 / 反馈（点踩）。
- **会员权益对比表**（免费 vs 会员）+ 自动续费「到期前 3 天短信通知」。
- **演示环境切换**（微信内 / 外部浏览器，标注仅演示）。
- **Token 超额拦截（0614）**：达上限时任何提问 / 语音 / 实时会话被拦截，顶部珊瑚红条放大抖动提醒。

**🔵 调整 / 修复**
- **登录整体重构**：H5 无法一键获取手机号（小程序专属）→ 统一「手机号 + 验证码」绑定；微信内置浏览器走官方授权弹窗（昵称/头像/性别/地区）、外部浏览器走 open.weixin.qq.com 扫码授权，回调后按「是否已绑手机号」决定引导绑定（见 PRD 图 9）。
- **个人资料裁剪修复**：「确定」无效根因＝原图 base64 撑爆本地存储 → 改 canvas 真裁剪小图；标题穿透遮挡已修；「取消」留在个人资料页。
- 外链跳转改黑底白字倒计时条、移到屏幕中上方，触发时不再关闭右侧「参考来源」抽屉。
- 手机号门槛文案改一行；会员中心权益名统一五项；我的永享加搜索 / 筛选 + 视频控件；多模态区标题改「相关媒体资源」+ 付费墙居中；流式答案仅新回答打字机；兑换码支持拍照 OCR。

**🟠 删除**
- 账号合并逻辑；落地页「浏览器打开？扫码登录」入口；扫码登录页二维码中央 Logo。
