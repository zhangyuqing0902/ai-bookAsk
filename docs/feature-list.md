# AI 问书 · 三端功能清单（开发依据）

> 版本：评审定稿 v1 · 日期：2026-06-01
> 范围：机构前台 H5 / 机构后台 Web PC / 平台超管 Web PC，三端高保真原型（只 mock，不接真后端）
> 说明：本清单为产品评审多轮确认后的定稿，作为原型开发的功能依据。每个功能点均含功能描述。

---

## 一、机构前台 H5（C 端用户）

### 1. 登录与权限

| 功能点 | 功能描述 |
|---|---|
| 登录（微信 / 手机号两种方式） | 落地页仅展示 AI 问书 icon + slogan + 两个登录按钮（不展示「你好我是…AI 版作者」问候语）。① 微信登录：授权获取 openid，登录成功后若该微信号未绑手机号则引导输入手机号 + 短信验证码绑定（提供「暂不绑定」可跳过直接进首页），已绑则直接进首页；若手机号在本机构已存在但未绑微信，通过**手机号 + 验证码**校验本人，通过则合并绑定，失败则清空并回到手机号输入步骤。② 手机号登录：手机号 + 短信验证码直接登录。 |
| 二维码失效落地页 | 二维码作废 / KP 下架时弹窗提示「二维码失效，即将跳转至 AI 问书首页」，倒计时 3 秒自动跳首页。 |
| 游客模式 | 未登录可浏览只读内容；创建会话、发送问题等需授权操作时引导登录，登录成功后自动补发游客态已输入问题；左下角「我的」未登录态点击亦触发登录。 |
| 登录态有效期 | 登录态有效期 30 天，每次操作自动重置有效期。 |

### 2. AI 会话

| 功能点 | 功能描述 |
|---|---|
| Agent 形象 sticky 区 | 会话顶部固定 Agent 形象，会话区下滚渐隐隐藏、上滚复原显示。 |
| Agent 路由（一次性判定） | 扫码或 H5 链接均带 KP ID；首次进入会话按 KP ID 拉绑定 Agent 渲染形象 / 声音，并在 slogan「有问题，就问 AI 问书」下方展示系统基于 KP 自动提炼的 3 条入门示例问题（点击即发送）；后续提问不再重判定。 |
| 流式答案 + 多模态卡片 | 文字打字机式流式输出；卡片内**文字区在上、媒体区（图 / 音 / 视）在文字区下方**，两区做明显视觉区分；媒体并列排布支持左右滑动，**顺序不固定，按后台返回的相关度从左到右排列**（非图→音→视固定序）；受限内容按权益打锁标；点击任一媒体可放大。 |
| 答案操作（复制 / 点赞 / 反馈） | 每条 AI 答案下方、「参考 N 篇知识」入口左侧并排展示三个操作 icon：复制（复制答案文本）、点赞、反馈（点踩 / 提交问题反馈）。 |
| 回答溯源富信息 | 答案下展示「参考 xx 篇知识」入口，点击后从右侧抽屉拉出溯源列表，**仅按相关度从高到低排序、不展示相关度数值**；每条含源头图（48×48）+ 知识 KP 名称 + 文件名称 + 章节页码；book 类型源头图为 KP 封面、web 类型为站点截图，二者尺寸 / UI 一致；字段过长尾部省略号截断。条目点击行为：book 类型跳转该 KP 的纸书购买链接，若链接为空则提示「无跳转链接」（3 秒后自动消失）；web 类型直接打开来源网页。 |
| 推荐追问 | 展示在多模态卡片下方，2–3 个追问**上下纵向堆叠（非左右并列）**，点击即发送。 |
| 多模态全屏 lightbox + 版权保护 | 放大后顶部固定「因版权限制，暂不支持保存下载」；底部缩略图条可点击切换，主预览区支持左右滑动浏览同卡片其他媒体。 |
| 实时电话式语音 | **会员专属能力**：未开通会员点击电话入口时引导开通；开通后方可进入。独立电话页：中央 Agent 形象，下方居中「挂断」（返回原会话并定位最近一轮），左侧「静音」（关闭时不收音），右侧「字幕」（开启时显示字幕、Agent 形象隐藏，关闭时反之）。挂断按钮上方展示实时语音状态文案 + 动画（如「正在听…」「正在回答…」）。字幕开启时对话文本按交互时序自上而下、左对齐排列，用户文本浅灰色、AI 回复深黑色。全程问答同步写入 AI 会话历史。 |

### 3. 付费墙与权益

| 功能点 | 功能描述 |
|---|---|
| AI 会员订阅页 | 仅展示首月折扣价、月度价两档，无年付；含自动续费协议入口与 72 小时宽限期说明。 |
| 永享购买 | 触发受限内容时弹出永享购买卡，展示对应图 / 音 / 视单价与权益说明（永久解锁该单条内容）。 |
| 兑换码兑换 | 「我的 - 兑换码」入口核销会员权益，支持手动键入与调起相机 / 相册 OCR 拍照识别，需获取摄像头 / 相册权限。 |
| 微信支付过场 + 结果页 | 唤起微信支付 → 等待回调 → 成功 / 失败结果页；成功页「返回会话」定位到支付前会话最近一轮对话。 |

### 4. 我的

| 功能点 | 功能描述 |
|---|---|
| 账户管理 | 仅保留「换绑手机号」（新号 + 验证码校验）。 |
| 用户协议 | 独立入口，纯文本 / Markdown 渲染。 |
| 隐私政策 | 独立入口，与用户协议并列。 |
| 联系客服弹窗 | 展示机构后台配置的一个二维码（企微 / 个微），下方电话、邮箱均非必填，有则展示无则隐藏并自动补位（不留空块）。 |
| 会员中心 | 非会员展示会员权益 + 立即开通入口；会员展示当前权益、到期时间、权益明细、「取消自动续费」入口（续费为自动，无手动续费按钮）。 |
| 我的永享 | 已购永享 KP 列表，可点击查看；放大预览图 / 音 / 视不支持保存下载，放大界面顶部固定版权保护文案。 |
| 我的订单 | 顶部四 Tab 切换：全部 / 会员 / 永享 / 兑换码；每单展示金额、状态、时间；仅「永享」订单展示关联 KP，会员与兑换码订单不展示 KP。 |

## 二、机构后台 Web PC（机构运营）

### 1. 登录与主控台

| 功能点 | 功能描述 |
|---|---|
| 登录页 | 按 design 视觉稿实现，账号 + 密码（+ 可选验证码）。 |
| 主控台 Dashboard | 关键指标速览（DAU、当前会员数、累计 GMV、累计提问数）+ 最近 7 日趋势图 + 快捷入口。 |

### 2. Agent 人设系统

| 功能点 | 功能描述 |
|---|---|
| Agent 列表 | 机构 Agent 全机构唯一，机构创建时平台自动初始化默认头像 / 形象 / 声音；普通 Agent 用户可自由创建 / 编辑 / 删除，无数量限制。 |
| Agent 详情编辑 | 头像 / 形象支持 png/gif/jpg/jpeg；TTS 参考音上传作声音克隆素材；回答 Prompt 富文本（受权限点控制）；右侧 Tab 展示该 Agent 已关联 KP 列表，仅可查看不可编辑，并提供「前往 KP 详情修改关联」跳转入口（跳转到指定 KP 详情的基础信息 Tab 调整 Agent 绑定）；入门示例问题由系统按关联 KP 内容自动提炼 3 条，动态生成。 |
| 回答 Prompt 编辑权限点 | 所有 Agent（内置 / 新建）均可被勾选该权限，默认不勾选；仅勾选后回答 Prompt 才可编辑。 |

### 3. 知识产品 KP

| 功能点 | 功能描述 |
|---|---|
| KP 列表（卡片布局） | 卡片展示封面图（9:16）+ KP 名称 + 来源标签（自建 / 共享）+ 状态 + 关键字段（关联 Agent、文件数、提问数等）；顶部「新建 KP」「导入分享 KP」按钮。 |
| 创建 KP 弹窗 | 输入 KP 名称等基础字段后创建；不在此选切片颗粒度（切片颗粒度针对知识库内单个文件，在上传文件时设定）。 |
| KP 详情 Tab 顺序 | 基础信息 → 知识库 → 永享 → 定价与权益 → 二维码 → 分享。 |
| 基础信息 Tab | 上传 KP 封面图（9:16，作 book 类型溯源图）；KP 名称、简介；纸书购买链接（非必填，问号 tooltip：「填写后，用户查看内容溯源时可跳转该链接」）；关联 Agent（下拉选本机构 Agent）；KP 状态以功能按钮切换：未发布 / 已发布 / 已下架 / 删除，发布、下架、删除均二次确认。删除为软删除（前期不做回收站，对用户等同永久删除）。仅「已发布」状态的 KP，其知识库内容才会被纳入用户问答检索；未发布 / 已下架 / 删除不参与检索。 |
| 知识库 Tab — 上传与浏览 | 无子 Tab，用户直接上传文件即可；列表支持按文件名（文档 / 图片 / 音频 / 视频）模糊匹配，支持按文件格式下拉筛选（文档 / 图片 / 音频 / 视频）。 |
| 知识库 — 文档（Word/PDF） | 支持批量上传；用户选切片方式（章节 / 语义【推荐】 / 字数）+ 两个独立勾选项「解析文档内图片」「解析文档内表格」（勾选则对应内容一同向量化）。 |
| 知识库 — 纯图片 | 仅单独上传；用户手动填写图片含义文字后向量化。 |
| 知识库 — 纯音频 | 仅单独上传；系统自动提取字幕（提取耗时，弹窗内显示进度条并提示「正在提取字幕，请勿关闭弹窗」）→ 用户确认 → 向量化。 |
| 知识库 — 纯视频 | 仅单独上传；系统自动提取视频音轨字幕（提取耗时，弹窗内显示进度条并提示「正在提取字幕，请勿关闭弹窗」）→ 用户确认 → 向量化。 |
| 知识库 — 问号说明 | 问号 tooltip 三行换行展示：1）支持格式：Word、PDF、图片、音频、视频；2）能力说明：可解析文档内图片、表格数据；3）上传要求：音频、视频需单独上传处理，文件支持批量。 |
| 文件处理状态 | 每个文件显示：上传中 / 解析中 / 切片中 / 向量化中 / 已向量化 / 向量化失败；失败仅支持重试、删除（无替换）；失败状态下鼠标悬浮显示失败原因。 |
| 向量化时机 | 文件上传即向量化（与发布解耦）；KP「已发布」前置条件为至少有一份文件「已向量化」，否则发布按钮置灰并提示原因。 |
| 永享 Tab | 列出该 KP 全部图 / 音 / 视条目，支持运营按条目设定永享单价（用户购买后永久解锁该单条内容）；顶部筛选区：按内容名称模糊匹配、按格式单选（图 / 音 / 视）、按价格状态单选（已设价 / 未设价）。 |
| 定价与权益 Tab | 基础权益标签：免费 / 会员（二选一互斥）；界面不常驻显示「二选一互斥」字样，在标签旁放小问号 ⓘ，鼠标悬浮展示互斥说明。 |
| 二维码 Tab | 以「二维码包」为单位的列表（一次创建 = 一个包）。列表字段：二维码包名称、权益模式、数量、操作（查看 / 下载整包全部二维码 zip）；支持按包名称模糊搜索、按权益模式筛选。创建二维码包填写：包名称（运营手动填写）+ 权益模式（无权益 / 首扫绑定，后扫引导）+ 生成数量——无权益时数量锁定为 1 且不可编辑，仅「首扫绑定，后扫引导」可填写数量。点击包进入详情页：① 首扫绑定，后扫引导包展示每个二维码的「是否扫描、扫描时间、首扫用户、后扫累计次数」；② 无权益包仅展示「二维码、累计扫描次数」两个字段。 |
| 分享 Tab | 「新建分享」为独立弹窗（与下方已分享链接列表分开）；弹窗内设：分享模式（实时同步 / 独立快照）、链接时效 1–15 天（默认 3 天）、访问人数 1–10 人或不限制（默认 1 人，限 N 人时被正确消费 N 次后即作废即使未到期）。已分享链接管理列表字段：链接 / 密码（列内含复制小 icon，点击提示「复制成功」并将链接 + 密码写入剪切板）、模式、时效、消费 / 上限、状态、操作（取消分享）；支持取消分享（取消后链接立即失效，但取消前已被正确消费的链接无法撤回）；放问号 icon 说明取消分享逻辑。 |

### 4. KP 跨机构分享

| 功能点 | 功能描述 |
|---|---|
| 导入分享 KP 入口 | KP 列表顶部「导入分享 KP」按钮，弹窗输入分享链接 + 密码；实时同步导入后直接在列表显示，独立快照导入后 KP 名称自动加「-快照」后缀；独立快照旁加问号说明：「独立快照为分享者发起分享那一刻的内容快照，非接收时刻」。 |
| 共享类型字段 | 每个 KP 标识「自建」或「共享」（被分享或接收他人分享的均标记共享），该字段在 KP 列表卡片呈现。 |
| 实时同步 KP 编辑限制 | 实时同步模式下接收方对该 KP 全部信息只可查看不可编辑（含基础信息、知识库、永享、定价与权益、Agent 绑定、二维码 Tab 等）；该 KP 在接收方机构可正常用于检索问答；接收方不可再次分享；分享仅 KP 创建机构账户可操作。 |

### 5. 运营中心

| 订单管理 | 三类订单（会员 / 永享 / 兑换码）筛选与导出；不做退款工单。 |

## 三、平台超管 Web PC（平台超级管理员）

### 1. 登录与主控台

| 功能点 | 功能描述 |
|---|---|
| 登录页 | 平台超管账号 + 密码登录。 |
| 主控台 Dashboard | 全平台关键 KPI（机构数、累计用户、GMV、提问总量等）+ 趋势图（去掉告警 KPI）。 |

### 2. 机构管理（含机构账户管理子菜单）

| 功能点 | 功能描述 |
|---|---|
| 机构列表 | 字段：序号、机构 ID、机构名称、状态（正常 / 停用）、上级机构（有则展示，无则 -）、LLM 配置（平台默认 / 自配厂商名称）、微信支付（未配置 / 已配置）、操作。 |
| 机构状态联动 | 正常：可正常登录后台、访问前台；停用：访问前后台均提示「机构服务已暂停，如需恢复请联系管理员」。 |
| 创建机构弹窗 | 弹窗字段：机构名称、上级机构（选填，建立层级关系，账户据此自动推断）、备注（选填）、状态（默认正常）、LLM 配置（平台默认 / 自配）。机构 ID 在创建后由系统自动生成、弹窗内不展示；不再有「出版方」字段（已统一为上级机构）。 |
| 机构详情 Tab 1 — 基本资料 | 基础信息查看与编辑（含上级机构）。 |
| 机构详情 Tab 2 — 机构配置 | 左侧二级 Tab 自上而下：LLM 配置 / 联网配置 / 微信支付配置。 |
| 机构详情 Tab 3 — 用量看板 | 平台视角单机构运营健康度看板，支持时间区间筛选 + 导出。详细指标见文末「用量看板指标表」。 |
| 机构账户管理（子菜单） | 置于「机构管理」菜单下。全平台账户列表字段：序号、账户 ID、账户名称、姓名、所属机构、上级机构、角色、联系方式、状态（正常 / 停用）、最近登录时间、创建时间、操作。 |
| 账户新建 | 弹窗字段：账户名称、姓名、所属机构（必选，上级机构由所选机构自动推断，无需单独选择）、角色（创建时即可指定，后期可变更）、联系方式、备注（选填）、状态（默认正常）。 |
| 账户状态联动 | 正常：可访问机构后台；停用：访问后台提示「账户服务已暂停，如需恢复请联系管理员」。 |

## 附：数据看板指标表（机构后台）

> 整个看板顶部统一时间区间筛选（今日 / 7 日 / 30 日 / 自定义），全 Tab 与图表联动；整页支持导出。

### 总览
| 指标 | 公式 | 业务意义 |
|---|---|---|
| 累计注册用户 | COUNT(DISTINCT user_id) | 规模 |
| 当前会员数 | 会员未到期的去重用户数 | 商业化基本盘 |
| 累计 GMV | SUM(订单金额 WHERE 状态=已支付) | 总营收 |
| 当日 DAU | COUNT(DISTINCT user_id WHERE 当日有登录或发问) | 即时活跃度 |

### 用户分析
| 指标 | 公式 | 业务意义 |
|---|---|---|
| DAU / WAU / MAU | 周期内有效行为去重用户数 | 粘性分层 |
| 新增用户数 | 当日首次注册数 | 拉新效果 |
| 次日 / 7 日 / 30 日留存率 | 新增用户在 N 日后仍活跃占比 | 产品价值是否站得住 |
| 来源分布 | 扫码进入 vs 直接访问 | 渠道效果 |
| 活跃时段分布 | 按小时统计提问数 | 运营 / 推送时点参考 |

### 提问分析
| 指标 | 公式 | 业务意义 |
|---|---|---|
| 总提问数 | COUNT(提问) | 使用强度 |
| 人均提问数 | 总提问数 / DAU | 单用户深度 |
| 平均会话轮次 | 总提问数 / 总会话数 | 对话沉浸度 |
| 受限内容触发率 | 触发付费墙次数 / 总提问数 | 付费墙强度是否合理 |
| 按 Agent 提问分布 | 每个 Agent 接收提问占比 | Agent 配置 / 路由质量 |

### 转化分析
| 指标 | 公式 | 业务意义 |
|---|---|---|
| 付费用户数 / 付费转化率 | 付费用户 / 累计用户 | 商业化核心 |
| 会员漏斗 | 看到会员页 → 点击购买 → 完成支付 三步转化 | 找漏斗瓶颈 |
| 永享转化率 | 购买永享数 / 触发永享付费墙数 | 永享定价是否合理 |
| ARPPU | 总收入 / 付费用户数 | 单付费用户产值 |
| 续费率 | 到期续费数 / 到期会员数 | 长期价值 |

### 热门 KP
| 指标 | 取数 | 业务意义 |
|---|---|---|
| 提问数 TOP 10 KP | 按 KP 维度统计提问 | 真正被使用的内容 |
| 付费转化贡献 TOP 10 KP | 通过该 KP 触发的会员 / 永享订单数 | 营销价值高的内容 |
| 永享购买 TOP 10 KP | 按 KP 永享订单数 | 永享潜力 |
