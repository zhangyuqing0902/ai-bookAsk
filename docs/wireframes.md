### 15.1 登录（落地页 + 表单，对应 §一、§十二）

- 落地页自上而下：知识核动效 → 「AI 问书」→ slogan「答案有出处 · 知识更可信」→ 底部按钮区。
- 底部按钮区（从上到下）：`[微信授权登录]`（绿）/ `[手机号登录]`（描边）/ 机构来源文案「知识由 中国医学临床百家 提供」/ 协议勾选行。
- **协议勾选**：单选框 + 文案「已阅读并同意《用户协议》与《隐私政策》」；《用户协议》《隐私政策》主题色高亮、点击打开对应协议 H5；点击**方框或文案任意处**均自动勾选；**未勾选时点登录按钮 → toast「请先阅读并同意…」(3s)**，不进入。

### 15.2 AI 会话（对应 §1.1 / §1.2 / §1.3 / §1.4）

- **欢迎态 slogan**：「有问题,就**问 AI 问书**」中「问 AI 问书」做打字机动效，从左至右循环。
- **输入栏**：真实输入框（光标、可输入，回车发送）；「深度思考 / 智能搜索」chips 可点击切换选中态；右下按钮——输入为空时为**语音按钮**，有文字时变为**发送按钮**。
- **短语音输入（非实时电话）**：点击/按住语音按钮（或长按输入框）→ 调起麦克风权限 → 录音浮层显示**音浪**（随音量起伏）+「松手发送,上滑取消」→ 松手转写为文字并自动发送。
- **答案区**：文字打字机逐字输出 → 完成后才显示多模态区；多模态区背景为**灰色系**（非黄）；标题「**相关媒体资源**」；含若干假数据，受限项显示锁。
- **答案操作行**：左 `复制 / 点赞 / 反馈(踩,点赞倒置)`；点赞与反馈**互斥**（点赞后再反馈则取消点赞，反之亦然）；点赞点击即填充，反馈**提交后**填充；右侧为「**参考 N 篇知识**」——**纯灰色文案按钮、无左侧 icon**，点击打开溯源抽屉。
- **反馈弹窗**（点「反馈」弹起，底部 sheet）：标题「反馈」+ 关闭；标签 `有害/不安全 · 虚假信息 · 没有帮助 · 其他`（可选）；多行文本框「我们想知道你对此回答不满意的原因…」；`[提交]`。
- **多模态 lightbox**：左右箭头切换、底部缩略图联动高亮；受限内容放大后**只显封面**，点封面 → 弹永享付费墙引导付费。
- **推荐追问**：假数据，点击即作为新问题发送。
- **新建会话**：右上 `[+]` 与左抽屉「新建会话」一致；若当前已是新会话（空）→ toast「已经在新对话中」(3s)。
- **左抽屉**：顶部搜索框；历史按时间**降序分组**（今天 / 昨天 / 年月）；每条标题＝该会话首个问题被 AI 总结的用户意图，超长省略号；页脚「头像 + 微信昵称 + 手机号 + 会员标签 + 更多 icon(›)」，点击进「我的」。
- **实时电话语音页**：挂断按钮下方一行灰字「内容由 AI 生成」。

### 15.3 我的（对应 §7 / §13）
