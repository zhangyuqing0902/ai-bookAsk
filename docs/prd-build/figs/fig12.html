图 12 · 免费 / 会员 / 永享 三场景权益流程
每个场景独立呈现「点击 → 判定 → 解锁 / 付费墙 → 购买 → 解锁」全路径；会员与永享两池独立
场景一点击「免费」内容
点击免费内容
word / pdf 或标「免费」的图音视
›
判定：免费内容
任意已登录用户
›
✓ 直接解锁
无需付费，直接查看 / 播放
场景二点击「会员」内容
点击会员内容
标「会员」的图 / 音 / 视
›
判定：是否 AI 会员？
只看会员身份，与永享无关
›
是 → ✓ 解锁
直接查看 / 播放
否 → ✗ 会员付费墙
引导开通 AI 会员
›
开通会员（微信支付）
返回原会话即解锁
›
✓ 解锁全部会员内容
会员有效期内通用
场景三点击「永享」内容
点击永享内容
设了永享单价的图 / 音 / 视
›
判定：是否已买断该 KP？
按「用户 + KP」粒度，与会员无关
›
是 → ✓ 解锁
永久持有、可反复查看
否 → ✗ 永享付费墙
按单价买断该单条内容
›
买断该条（微信支付）
返回原会话即解锁
›
✓ 永久解锁该条
跨 KP 不通用
两池独立（关键规则）：「会员」与「永享」是两条相互独立、互不替代的解锁维度——开通会员不会解锁未买断的永享内容；购买永享不会解锁会员内容。例：会员用户点击永享内容仍需按单价买断；永享买断用户点击会员内容仍需开通会员。
